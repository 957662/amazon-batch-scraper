### 列定义

| 列 | 字段名 | 数据类型 | 格式 | 列宽 | 说明 |
|---|--------|----------|------|------|------|

| I | 评论数 | 数字 | #,##0 | 12 | 8,421 |
| J | 库存 | 文本 | 条件格式 | 10 | 绿色=有货，红色=缺货 |

| L | BSR 排名 | 数字 | "#"# | 10 | #306 |

| P | 采集时间 | 日期时间 | yyyy-mm-dd hh:mm | 18 | 2026-03-15 05:00 |

### 格式设置

**表头行 (第 1 行):**
- 字体：加粗
- 背景色：#4472C4 (蓝色)
- 字体颜色：白色
- 边框：底部双线
- 冻结：冻结首行

**数据行:**
- 字体：常规
- 边框：细线
- 行高：根据内容自动调整 (最小 20)
- 斑马纹：隔行变色 (白色/#F2F2F2)

**条件格式:**

1. **库存状态 (J 列):**
   - "In Stock" → 绿色背景 (#C6EFCE)
   - "Out of Stock" → 红色背景 (#FFC7CE)
   - 其他 → 黄色背景 (#FFEB9C)

2. **折扣力度 (G 列):**
   - >50% → 红色字体，加粗
   - 30-50% → 橙色字体
   - <30% → 常规

3. **评分 (H 列):**
   - ≥4.5 → 绿色字体
   - 4.0-4.4 → 橙色字体
   - <4.0 → 红色字体

4. **BSR 排名 (L 列):**
   - ≤100 → 绿色背景
   - 101-1000 → 黄色背景
   - >1000 → 常规

**图片嵌入:**
- 主图显示在 M 列
- 图片大小：60x60 像素
- 单元格对齐：居中
- 保持纵横比

## 🔧 自动化功能

### 公式

1. **折扣百分比 (G 列):**
   ```
   =IF(F2>E2, (F2-E2)/F2, "")
   ```

2. **市场份额 (品牌分析 H 列):**
   ```
   =B2/SUM($B$2:$B$100)
   ```

3. **平均价格:**
   ```
   =AVERAGEIF(条件范围，条件，价格范围)
   ```

### 条件汇总

- 总商品数：`=COUNTA(B:B)-1`
- 平均价格：`=AVERAGE(E:E)`
- 平均评分：`=AVERAGE(H:H)`
- 有货商品：`=COUNTIF(J:J,"In Stock")`
